- Per eseguire QuotesFinder è necessario installare Java sul proprio computer, il download è disponibile al seguente link: "https://www.java.com/it/download/"

- Dopo aver installato Java, per lanciare QuotesFinder è sufficiente aprire il file "QuotesFinder.jar"

- Il file delle stop words è modificabile in: "QuotesFinder > STOP WORDS > stop_words.txt"

- I file di testo vanno posizionati in: "QuotesFinder > TEXTS"
