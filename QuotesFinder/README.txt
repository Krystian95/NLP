- Per eseguire QuotesFinder è necessario installare Java sul proprio computer, il download è disponibile al seguente link: "https://www.java.com/it/download/"

- Dopo aver installato Java, per lanciare QuotesFinder è sufficiente aprire con un doppio click il file "QuotesFinder.jar". In alternativa è possibile lanciare QuotesFinder dal terminale posizionandosi nella directory contenente il file JAR e lanciando il comando: java -jar "QuotesFinder.jar"

- Su macOS potrebbe comparire un avviso per autorizzare l'esecuzione di QuotesFinder, procedere dando il consenso e, se necessario, premere il pulsante "Apri comunque" in "Preferenze di Sistema > Sicurezza e Privacy > Generali"

- Il file delle stop words è modificabile in: "QuotesFinder > STOP WORDS > stop_words.txt"

- I file di testo vanno posizionati in: "QuotesFinder > TEXTS"

- Le cartelle "temp_files" e "final" contengono i file necessari a QuotesFinder durante la sua esecuzione e vengono popolate ad ogni esecuzione del software
